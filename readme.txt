Plugins helps you to create store locations on the google map . 

== Description ==

Easy Store Locator allow you to mark store locations on the google map and with the help of shortcode you can display those locations along with the google map on the wordpress pages/posts.

Give **Easy Store Locator** a try. We are sure you will like it. 


= How to add location in the google map? =

1. First create a signup on [easystorelocator](https://easystorelocator.com/signup "easy store locator").
2. After successful account creation login to your control panel.
3. CLick the add location tab.
4. Finally click import




== Installation ==

1. Download Easy Store Locator plugin.
2. Upload the easy-store-locator folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress and Enjoy.
